# HotPotQA Baselines

Store blessed benchmark result JSON files here when a TERAG retrieval implementation is accepted as the baseline for a tier.

Suggested files:

- `terag_ppr_openai_smoke.json`
- `terag_ppr_openai_dev_5pct.json`
- `terag_ppr_openai_full.json`

Generated run outputs belong in `benchmarks/hotpotqa/results/` and are ignored by git.
